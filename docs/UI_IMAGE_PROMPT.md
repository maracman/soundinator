# Sound Studio Explore UI Mockup Prompt

Use case: ui-mockup
Asset type: full-screen web app interface mockup for an interactive generative synthesiser

Primary request: Design a dense but elegant single-viewport desktop interface for "Sound Studio" Explore Mode. The interface must expose many controls without requiring scrolling and should rely on compact visual cues, mini charts, icon-like buttons, and statistical distributions rather than explanatory paragraphs.

Functional content to represent: transport bar with Play, Stop, Randomise, Seed, Tempo; real-time waveform visualiser; live counters for Motifs, Sequence, Notes; Scale area with 12-tone/N-EDO toggle, preset dropdown, note grid with Off grey, In scale blue, Sub-scale gold, and Sub-scale weight; Melody controls with Interval shape, Interval range, Precision, Precision range and a small interval probability distribution chart; Root Pull controls with root note grid, pull strength/shape and a pull-over-phrase distribution chart; Register controls with centre, width, skew and a skewed bell curve; Rhythm controls with beat divisions, on-beat, off-beat, same length; Voice controls with vowel chips ah/ee/oo/eh/oh, formant change, Formant/Sine/Triangle segmented toggle; Surprise controls with probability, dimension checkboxes for Pitch, Octave, Formant, Rhythm, Dynamics, Rest, and incorporation; Percussion controls for Beat tick, Motif accent, Downbeat with sound selectors and volume sliders; Motif Repertoire controls for motif count, motif length, sequence probability, motif surprise; Rating and Presets with rating slider, preset name/save, My Presets tab and Shared Library tab.

Layout: no scrolling; 1440x900 dashboard. Top compact transport strip, central large visualiser and counters, left column for Scale/Voice, middle grid for Melody/Root/Register/Rhythm distributions, right column for Surprise/Percussion/Motif Repertoire/Presets. Use cards only for top-level panels, not nested cards.

Style/medium: high-fidelity modern web app UI mockup, crisp typography, restrained professional audio-tool feel, compact control density.

Color palette: dark graphite background, slightly lighter panels, amber primary actions, blue note grid active state, gold sub-scale state, green root notes, muted grey inactive controls. Avoid purple/blue gradient dominance, decorative blobs/orbs, stock imagery, huge hero sections.

Text constraints: include only short labels such as "Sound Studio", "Explore", "Play", "Stop", "Randomise", "Scale", "Melody", "Root Pull", "Register", "Rhythm", "Voice", "Surprise", "Percussion", "Motifs", "Sequence", "Notes", "My Presets", "Shared Library". Do not include "Phase 0", "research", "experiment", "study", or long instructional text.

Visual cues: show histogram bars, probability curves, note grids, small sparklines, waveform, compact counters, segmented toggles, chips, sliders, checkboxes.

Constraints: everything visible in one viewport, no vertical scrollbar, no lorem ipsum, no watermark.
